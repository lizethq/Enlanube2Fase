Ticket Number on Timesheet

A module to automatically append the helpdesk ticket number to the time recording entry.

If several tickets from the helpdesk module are assigned to a task, the time overview of the task does not show which ticket they come from. If a customer now wants to know to which tickets the time bookings belong, this is only possible with great manual effort. In order to address this problem, the ticket number is automatically appended to the description when the time is booked in a ticket. Thus, the ticket number is automatically displayed in the time overview of the task in the project module and thus also traceable for the customer.
